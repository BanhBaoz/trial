About Sensors

Sensor Instruction

  1.  Ensure that the microSD card is inserted.

  2.  Ensure that the light sensor on top of the box is not covered with anything. It shall receive the amount of sunlight similar the sunlight directed at the examined plant.

  3.  Gently insert the soil moisture module into the pot, so that no cables get disconnected.

  4.  Plug the sensor into the electrical network using USB-mini cable.

  5.  The microcomputer is starting, yellow light indicates it.

  6.  While the particular modules are switching on the light remains yellow. If something went wrong the light would turn red. In that case try to replug the microcontroller.

  7.  When all the modules have started properly, the sensor writes the modules’ states to a file. If this process succeeds, the light turns green, then everything is working properly. However, if the file couldn’t have been opened the light turns orange.

  8.  Leave the sensor for any amount of time desired.

  9.  Unplug the sensor from the power source when finishing taking data.

  10.  Take the soil moisture module out.

  11.  Uncover the lid of the box and take the microSD card out. The file with the data from the measurements is available there.

Additional Information

The sensor is a device assembled to collect data about a plant’s environment in-real time to determine its perfect living conditions. The most crucial factors influencing a plant’s state are: soil moisture, air humidity, ambient temperature and irradiation.

A sensor was built with the following modules:

  • Arduino Nano - base of the sensor: It is equipped with an ATmega328P microcontroller which is the brain of our board and CH340 chip. It has 32kB of Flash memory, 2kB of RAM.

  • AHT20 – temperature and humidity sensor I2C: It allows measuring temperature from -40°C to 85°C with an accuracy of 0,3°C and humidity with an accuracy of 2%. It is connected to the board via VIN and GND for a power supply and SCL and SDA for signal transmission.

  • BH1750 – light intensity sensor: This sensor operates in the range: 1 – 65535 lx scale with resolution up to 1 lx. It is connected to the board via VIN and GND for a power supply. For the signal transmission the module uses SDA and SCL which are connected to analogue pins on the microcomputer.

  • Soil moisture sensor: Allows to determine most crucial factor in analysis of the plant environment. It returns the information using analogue input on the board, giving us the information in range from 0% to 100% about the soil moisture. It uses the same power supply method as previous modules.

  • Micro SD Card Reader Module: Using this device all of our measurements on micro SD cards can be saved and stored and analysed later. It uses 5V VCC for power supply and four additional pins for transferring data from the board.

  • LED RGB diode module: It is going to communicate the sensor’s states to the user by flashing different colours (further described in the instruction). It uses 5V VCC and GND pins for a power supply and digital output pin for data transmission.

  • RTC DS-1302 – real time clock: Enables the sensor to point an exact date and time a measurement is taken. It has to be power constantly with the CR2032 battery. This module is connected to the board over VCC and GND for a power supply and SCL and SDA pins for data transfer.

The device is powered by a USB-mini cable connected to an Arduino board, so it’s very easy to set it at any place desired.
